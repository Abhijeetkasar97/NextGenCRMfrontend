# MUNextGenCRM - Frontend

This is the frontend part of **MUNextGenCRM**, a modern, professional CRM application built using **React**, **Redux Toolkit**, and **Vite**.  

## Features

- User authentication & role-based access
- Dashboard with analytics charts
- Lead and activity management
- Notifications and real-time updates (Socket.io)
- Modular and reusable components
